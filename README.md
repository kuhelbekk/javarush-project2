#  **Island**
____
### Description:
#### The program is a console application. 

application settings are in the file `resources/islandOne.json`

получилось заставить каждое животное двигаться есть и размножатся в разных потоках.
Каждое животное ходит независимо.

Животные хранятся в общем списке острова, и в списке каждой клетки
также животное знает о клетке, а клетка знает о острове и своей позиции на нем
убрать связность кода до конца не получилось, так как в острове хранится список всех животных острова

животное теряет "сытость" ночью, при перемещении и при размножении
- `class UserDialog` нужен для задания настроечного файла острова в формате json
- `class IslandStat` выводит статистику острова, в подробной статистике планировал использовать emoji но в моей консоли они не рисуются
- `class ParamIslandLoader` загружает из файла тип животных
- `class AnimalType` содержит данные животного, например максимальное количество на клетку и эмоджи, а также клас животного для клонирования.
- `class Cell` содержит описание клетки, свою позицию на острове(решение плохое, возможно перепишу на NavigatibleMap in Island ) массив животных и ссылку на растения. Содержит методы проверки максимального количества животных и поиска еды
- `class Island` содержит клетки и общий список животных, в нем запускается процедура islandDay() которая отвечает за сутки. У списка животных в FixedThreadPool вызываются  методы eat и move, размножение идет в основном потоке, к сожалению не успел дописать размножение красиво. В функции islandMorning() растет трава и животные набираются сил для передвижения. метод getPossibleDirectionCell() предлагает массив клеток куда можно пойти.

Архтектрура программы позволяет в последствии легко реализовать:
 - добавление новых животных с уникальным поведением
 - Клетки заполненные водой или плазменной субстанцией.
 - Несколько островов с уникальными животными.
 - Остров с шестиугольными клетками.
